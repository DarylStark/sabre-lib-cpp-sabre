Classes to override
===================

To implement your own implementation of the Sabre library, you have to override the following classes and methods:

-   ``sabre::core::Factory``
-   ``sabre::devices::RgbPixelStrip``
-   ``sabre::hal::Gpio``
-   ``sabre::hal::InputGpio``
-   ``sabre::hal::OutputGpio``
-   ``sabre::hal::Serial``
-   ``sabre::net::HttpServer``
-   ``sabre::net::MqttClient``
-   ``sabre::net::WifiSoftAp``
-   ``sabre::net::WifiStation``
-   ``sabre::os::Queue``
-   ``sabre::os::Service``
-   ``sabre::platform::Platform``
-   ``sabre::time::NtpClient``
-   ``sabre::time::WallClock``
-   ``sabre::utility::WaitFor``
